# 网页计算器插件

这是一个Microsoft Edge浏览器插件，允许用户在浏览网页时直接使用侧边栏进行简单的数学计算。

## 功能特点

- 双击网页上的数字自动添加到计算器中
- 通过侧边栏显示和操作，保持在浏览器上方
- 支持基本的算术运算：加、减、乘、除
- 提供键盘快捷键支持
- 选中数字时提供视觉反馈

## 安装方法

1. 打开Microsoft Edge浏览器
2. 在地址栏输入 `edge://extensions/` 并按Enter
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本插件的文件夹
6. 插件将被安装并显示在扩展栏中

## 使用说明

1. 点击插件图标或使用快捷键 `Ctrl+Shift+K` 打开侧边栏
2. 浏览网页时，双击任意数字文本将其添加为第一个计算变量
3. 双击另一个数字将其添加为第二个计算变量
4. 在侧边栏中点击操作符按钮(+、-、×、÷)或使用键盘输入对应的符号
5. 点击"计算"按钮或按Enter键执行计算
6. 计算结果将显示在侧边栏中
7. 点击"重置"按钮或按Esc键清空所有变量

## 键盘快捷键

- `+`: 选择加法运算
- `-`: 选择减法运算
- `*`: 选择乘法运算
- `/`: 选择除法运算
- `Enter`: 执行计算
- `Esc`: 重置计算器
- `Ctrl+Shift+K`: 打开/关闭侧边栏

## 文件结构

- `manifest.json`: 插件配置文件
- `background.js`: 后台脚本，处理消息和侧边栏逻辑
- `content.js`: 内容脚本，注入网页并监听双击事件
- `sidebar.html`: 侧边栏UI界面
- `sidebar.js`: 侧边栏交互逻辑
- `icon.svg`: 插件图标源文件

## 注意事项

1. 插件当前使用icon.svg作为图标文件，无需单独创建PNG图标文件
2. 插件只能识别纯数字或主要包含数字的文本
3. 支持整数、小数和负数
